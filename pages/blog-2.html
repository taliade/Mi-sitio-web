¿QUE ES HTML5?

HTML5 es la última versión del lenguaje de marcado de hipertexto HTML (Hypertext Markup Language), utilizado para crear y diseñar páginas web. Se lanzó en 2014 y es una actualización importante del lenguaje HTML.

HTML5 tiene varias mejoras y características nuevas en comparación con versiones anteriores de HTML, como la capacidad de reproducir audio y video directamente en el navegador sin necesidad de complementos como Adobe Flash Player. También incluye nuevas etiquetas para estructurar mejor el contenido web, como "article", "header", "footer", "nav" y "section", entre otras.

Otras características de HTML5 incluyen la capacidad de crear aplicaciones web offline y la mejora de la accesibilidad web para personas con discapacidades visuales o auditivas. HTML5 también ofrece una mejor compatibilidad con dispositivos móviles y tabletas, lo que permite una experiencia de navegación más uniforme en diferentes dispositivos.
